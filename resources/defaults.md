# defaults.json (mapping file)

`defaults.json` is strict JSON (no comments). See this file for documentation.

- `urls`: URL scheme -> bundle identifier
- `types`: UTI -> bundle identifier

Schemes are without a colon (http not http:).

Configured identifiers:
- Package identifier: com.yourorg.defaultappctl
- One-time GUI retry label: com.yourorg.defaultappctl.once
